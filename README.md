# RGB LED

**RGB LED** project shows how to control a RGB LED

Author: **Huy Dang**

### Component Required

- (1) x Elegoo Uno R3 
- (1) x Bread board (4) x M-M wires- (1) x RGB LED- (3) x 220 ohm resistors
